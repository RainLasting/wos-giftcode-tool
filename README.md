# Whiteout Survival 礼包码兑换工具 (WOS Gift Code Tool)

***

本项目 fork 自 [justncodes/wos-giftcode](https://github.com/justncodes/wos-giftcode)（v4.0.0）

***

## 新增内容

### GUI 图形界面

- **三栏布局**：主面板 + 左侧玩家列表面板 + 右侧礼包码列表面板
- **侧边栏自动展开/折叠**：启动时自动展开，支持手动切换
- **兑换设置面板**：礼包码输入框（支持回车触发）、开始/停止按钮
- **实时进度面板**：进度条 + 百分比显示 + 成功/已兑换/失败计数器
- **汇总报告弹窗**：兑换完成后弹窗展示完整统计

### 玩家 ID 管理系统

- **添加玩家**：输入框 + 回车/按钮添加新玩家 ID
- **编辑玩家**：左键点击玩家条目弹出编辑对话框
- **删除玩家**：右键菜单删除，带确认对话框
- **自动名称更新**：兑换过程中自动从 API 获取玩家昵称并回写 CSV
- **列表实时刷新**：增删改操作后自动刷新显示

### Wiki 礼包码爬取

- **爬取**：Official Wiki（whiteoutsurvival.wiki）
- **一键填入**：爬取到的礼包码以按钮形式展示，点击即可填入兑换框
